***  Project Overview**

This project is developed as part o8f the Flipped Course requirements at VIT Bhopal University. The objective is to design a simple and practical Python-based system that helps students find suitable transportation options while traveling from VIT Bhopal to Bhopal City.
Students often face difficulty in finding autos, cabs, or buses for outstation travel, and this system provides quick access to transport details.
To create a Python program that:
-Lists autos, cabs, and buses available from VIT Bhopal.
-Provides price and contact information.
-Offers a search feature for quick filtering.
-Displays emergency contact numbers.
-Ends execution after one operation.
-Shows a polite closing message.

 **Features Implemented**

1) Auto Options
Shows available auto drivers, prices, and contact numbers.

2)Cab Options
Includes cab categories like Swift, Ertiga, Omni Van, etc.

3) Bus Options
Displays charted bus routes, timings, and fares.

4)Search Feature
Users can search transportation by:

Vehicle type (e.g., Swift, Ertiga)
Route (e.g., ISBT)
Time (e.g., 6 PM)

5)Closing Message
After every output, users see: "Thank you, Have a Safe Journey!"
